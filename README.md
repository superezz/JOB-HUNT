# 🚀 AI Job Automation Platform

An AI-powered platform that helps users:

- Analyzes projects using AI (Gemini)
- Extracts skills automatically
- Stores user skill data
- Match jobs based on skills
- Builds a smart developer profile
- Generate ATS-optimized resumes
- Download resumes as PDF

## 🔥 Features
- AI Project Analyzer
- Skill Aggregator Engine
- Job Matching System
- Resume Generator (AI)
- PDF Download

## ⚙️ Tech Stack

- Frontend: React + Tailwind
- Backend: Node.js + Express
- Database: MongoDB
- AI: Google Gemini API
