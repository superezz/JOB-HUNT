# 🚀 AI Job Automation Platform

An AI-powered job assistant that:

- Analyzes projects using AI (Gemini)
- Extracts skills automatically
- Stores user skill data
- Builds a smart developer profile

## 🧠 Features

- 🔐 Authentication (JWT)
- 🤖 AI Project Analyzer
- 📊 Skill Aggregation Engine
- 🌐 Full Stack (React + Node + MongoDB)

## ⚙️ Tech Stack

- Frontend: React + Tailwind
- Backend: Node.js + Express
- Database: MongoDB
- AI: Google Gemini API

- Job Matching Engine
